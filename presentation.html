The Vision Agent Loop

Every complex task follows a disciplined Plan → Act → Verify cycle

📋

Plan

Analyze screenshot.
Form numbered strategy.
Explain reasoning.

→
⚡

Act

Execute ONE action.
Click, type, scroll.
Wait for UI to settle.

→
✅

Verify

Capture fresh screenshot.
Confirm action succeeded.
Continue or finish.

↩ loop until done=true (up to 25 steps)

Amazon Nova Integration

Component Model Capability
Intent Interpreter Nova 2 Lite Text reasoning — corrects garbled STT using screen context clues
Task Router Nova 2 Lite Classification — SIMPLE (instant) vs ADVANCED (agent loop)
Simple Executor Nova 2 Lite Structured output — natural language → executable JSON command
Vision Agent Nova Pro Multimodal — screenshot analysis, spatial reasoning, action planning

Key Technical Innovations

📐

Resolution Independence

Normalized [0,1000]² coordinate space. Works on any display — 13" MacBook to 27" Studio Display — without recalibration.

🔄

Cycle-Aware Stuck Detection

Detects repeating action patterns of any length (1-6), injects corrective prompts, and hard-bails after 3× repetition.

⚡

Hybrid Model Routing

Three-tier Nova hierarchy. Simple commands resolve in <1s via Lite. Complex tasks use Pro's full vision loop.

🎯

Edge Ruler Annotation

Minimal tick marks along screen edges give spatial reference without cluttering the screenshot the model sees.

📸

Conversation Windowing

Sliding window over history keeps system prompt + initial plan + recent 3 exchanges. Older screenshots pruned to maintain focus.

✨

Native macOS Overlay

Custom NSPanel with animated borders across all Spaces. Self-suspends during capture so the agent never sees its own UI.
