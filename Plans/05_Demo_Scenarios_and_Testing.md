# 🧪 "Do Anything" Hackathon Demos

These are the three core tests to validate the architecture and record for the submission video. Zero mockups allowed.

## 1. The Commerce Run
* **Objective:** Order a physical product from an unstructured e-commerce site via voice.
* **Challenge:** Handle a mid-checkout voice interruption (e.g., "Wait, change the shipping address to home").
* **Success Metric:** Agent successfully halts its current trajectory, corrects the text field, and proceeds.

## 2. The Subscription Guillotine
* **Objective:** Cancel a gym membership navigating a "Dark Pattern" UI.
* **Challenge:** Ignore massive, highlighted "Keep Membership" buttons and locate low-contrast, hidden "Confirm Cancellation" text.
* **Success Metric:** Visual extraction correctly identifies the manipulative UI and clicks the semantic target.

## 3. Cross-App Data Synthesis
* **Objective:** Read an email, open an attached PDF, and write a Slack message based on the content.
* **Challenge:** Multi-step reasoning across different OS windows.
* **Success Metric:** Seamless application switching and accurate context synthesis without losing state.
