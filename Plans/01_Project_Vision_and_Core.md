# 👁️ Sai: Project Vision & Core Features

## The Vision
We are building a Zero-Click Web experience. Moving beyond brittle APIs and blind voice assistants, Sai uses real-time multimodal reasoning to see the screen and operate the OS exactly like a human. 

## Core Disruptive Features
1. **True "Zero-API" UI Navigation:** Dynamically maps DOM and pixels in real-time, bypassing CAPTCHAs and anti-bot measures visually.
2. **Full-Duplex Interruption Architecture:** Uses Gemini Live API via WebSockets for mid-action interruption (e.g., stopping the mouse mid-move to change a prompt).
3. **Google Antigravity Orchestration:** Manages memory, tool-calling state, and task trajectory for long-horizon OS operations.
4. **Transparent Thought Stream:** A UI overlay exposing the internal monologue (Vision -> Audio -> Action) for ultimate transparency.

## Judging Criteria Alignment
* **Maximum Innovation (40%):** Shattering the text-box paradigm with OS-level command.
* **Technical Excellence (30%):** Pushing Gemini 3.1 Pro to its limits with a decoupled audio/visual loop.
